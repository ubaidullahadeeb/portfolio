# FashionX - Fashion E-commerce Website

## Overview

FashionX is a frontend fashion e-commerce website created to provide a modern online shopping experience. It includes separate sections for men's and women's fashion, shoes, handbags, and accessories. The website is fully responsive and designed with a clean and user-friendly interface.

## Features

- Responsive layout
- Product categories
- Product details
- Shopping cart
- Checkout page
- Search option
- Simple navigation
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Type

Frontend Website
